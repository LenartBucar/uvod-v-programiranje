### Kratke in preproste funkcije *lahko*<br>zapišemo z **anonimnimi funkcijami**

```
def moj_polinom(x):
    return x ** 2 - 2
```

```
>>> bisekcija(moj_polinom, 0, 2)
1.414213562355144
>>> bisekcija(lambda x: x ** 2 - 2, 0, 2)
```
